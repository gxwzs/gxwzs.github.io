markdown基本语法

标题

#号后接空格加文字，1个#号为一级标题，最高支持六级

字体

左右两边各一个*中间的文字为倾斜部分， 斜

左右两边各两个*中间的文字为加粗部分，粗

左右两边各三个*中间文字为倾斜加粗部分，

左右两边各两个~中文字为加删除线部分，删除

分割线

三个或三个以上的*，-都可以，示例

图片

！[图片下方文字]（图片地址 “图片title”）

支持本地与网络地址，本地图片可直接上传，title可不加

全英文输入符号

超链接

[超链接名]（超链接地址”title”）支持本地

列表

无序列表：用* - +加上空格接文字，示例：

  • 列表内容

有序列表：数字加点加空格接文字，示例：

  1. 列表内容

列表嵌套：上一级与下一级之间加tab，示例：

  • 列表内容
    1. 次级列表
      • 次次级列表

表格

用|号隔开单元格

示例：

表头 表头
内容 内容

代码

单行代码用反引号``

代码块用前后三个反引号``````

printf("hehe");

引用

用>号加上文字内容(可以多个>),示例:

引用内容
